AdvanceSCAN Roms Manager

AdvanceSCAN is a command line rom manager for MAME, MESS, xmame, AdvanceMAME, AdvanceMESS and Raine.

It runs in Linux, Mac OS X, DOS, Windows and in all the other Unix like platforms.

The official site of AdvanceSCAN is:

http://advancemame.sourceforge.net

This package contains:
advscan Rom manager
advdiff Diff utility for rom list files
